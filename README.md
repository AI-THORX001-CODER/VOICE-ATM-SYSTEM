

📌 Voice ATM System (Python)

A smart and interactive **Voice-based ATM system** built using Python that allows users to perform banking operations through voice commands or text input. This project simulates real ATM functionalities such as balance inquiry, cash withdrawal, deposit, and transaction history, enhanced with speech recognition and text-to-speech technology.

The system is designed to improve accessibility and user experience by enabling hands-free interaction, making it especially useful for beginners exploring **AI, automation, and voice assistants**.

---

🚀 Features

* 🎙️ Voice command recognition (Speech-to-Text)
* 🔊 Audio response using Text-to-Speech
* 🔐 Secure PIN authentication system
* 💰 Balance check, deposit, and withdrawal
* 📜 Transaction history tracking
* 📩 SMS-like alerts (simulated)
* ⌨️ Text mode fallback if microphone is not available

---

🛠️ Technologies Used

* Python
* SpeechRecognition
* Edge-TTS (Text-to-Speech)
* AsyncIO
* File Handling

---

 🎯 Project Objective

To create a beginner-friendly **AI-powered banking simulation** that demonstrates how voice technology can be integrated into real-world applications like ATMs.
